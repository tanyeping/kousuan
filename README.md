*  纯网页方案实现一个口算题目生成器，直接生成可用的html文件，要求如下： 
*  设置题目数量，根据设置的数量自动按A4页面部署口算题，并预留位置填写答案，可以通过鼠标或上下左右方向键移动答题位置  
*  可以支持加减乘除混合计算，可设置多个数据（1～10个）用于口算的随机数   
*  除法要保证能整除    
*  参与计算的多个数据均可以设置数据范围，题目在这个范围内随机生成   
* 以独立设置多个数据间的计算方式：加、减、乘、除 或随机   
* 可以设置答题时间用于倒计时   
* 生成题目可以支持打印成文档   
*  用户后点击启动按钮开始答题，并在右上角用比较大的字体和鲜艳的颜色显示倒计时时间   
*  计时结束，自动交卷，并给出成绩悬浮在答卷上方  |
*  卷后可以查看计算结果，并用于确认哪些题目计算错误   |
*  题的历史可以缓存到本地通过历史按钮进入可查看到某次答题的时间、时长和成绩   
*  要独立的按钮：题目设置（包含所有的设置项）、生成题目、启动答题、交卷、打印题目、查看历史按钮，其中启动答题放在左上角  
*  布局要清新雅致，显得高级，配色简介鲜艳适合小学生  |
*  题提交后，如果得分大于90，动画显示撒花和恭喜，并在历史记录中通过鲜艳的字体颜色或鲜艳的形式标记此次得了高分   
*  题时间设置单位为分钟  
*  置窗口只在点击设置按钮时出现，答题时收缩到左侧
